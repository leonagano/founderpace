User Profile — PRD - edit

Profile Requirements


Add a menu item, where user can edit their profile.

Fields
	•	name (editable)
	•	startup_name  (editable)
	•	profile_image (editable)
	•	socials: x_handle, linkedin, instagram, website (editable)

Behaviour
	•	User can edit socials and startup_name only.
	•	No username, no bio, no follower system.
